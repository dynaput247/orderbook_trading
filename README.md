# orderbook_trading
## Running
### Server
1. Install dependencies
   ```
   cd server
   yarn install
   ```
2. Start the server
   ```
   yarn dev
   ```
3. Server will start listening on port `8080`

### Client
1. Install dependencies
   ```
   cd client
   yarn install
   ```
2. Start the client
   ```
   yarn start
   ```
3. Client can be accessed by pointing your browser to `http://localhost:3000`.
